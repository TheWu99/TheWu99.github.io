安全边界

GitHub 个人 Pages 本身是公开静态网站，无法原生设置为“仅本人访问”。这里采用内容级保护：公开页面只有阅读器代码，私有正文必须使用你的 GitHub 只读凭证才能加载。

怎样创建最小权限的只读令牌？
  1. 打开 GitHub Settings → Developer settings → Personal access tokens → Fine-grained tokens。
  2. Resource owner 选择 TheWu99。
  3. Repository access 只选择 ChatGPT-Knowledge-Base。
  4. Repository permissions 只授予 Contents: Read-only；其他权限保持 No access。
  5. 设置较短有效期。令牌只粘贴在本页，不要写入任何公开仓库。
